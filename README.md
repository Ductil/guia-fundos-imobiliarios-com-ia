# Guia Completo de Investimento em Fundos Imobiliários (FIIs)

Este projeto foi desenvolvido como parte de um desafio prático do bootcamp da DIO, com o objetivo de utilizar a Inteligência Artificial como ferramenta de aprendizagem ativa.

A partir da criação de um caderno temático no NotebookLM, foram reunidas fontes abertas sobre Fundos Imobiliários (FIIs), definindo objetivos de estudo, elaborando perguntas estratégicas e testando variações de prompts para organização e síntese do conhecimento.

## Objetivo do Projeto
Criar um miniguia introdutório sobre Fundos Imobiliários, reunindo conceitos fundamentais, resumos estruturados, glossário financeiro e prompts reutilizáveis que apoiem revisões futuras.

## Ferramentas Utilizadas
- NotebookLM
- Inteligência Artificial Generativa
- GitHub
- Markdown

## Resultados
- Curadoria de fontes abertas sobre FIIs
- Resumos estruturados dos principais conceitos
- Glossário de termos financeiros
- Conjunto de prompts reutilizáveis para estudo e revisão
- Organização do conhecimento com apoio de IA

Este repositório tem caráter educacional e visa demonstrar o uso da IA como apoio ao aprendizado crítico e estruturado.
