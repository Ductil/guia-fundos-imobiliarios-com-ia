# Guia Completo de Investimento em Fundos Imobiliários (FIIs)

Este projeto foi desenvolvido como parte de um desafio prático do bootcamp da DIO, com foco no uso da Inteligência Artificial como ferramenta de aprendizagem ativa.

A partir da criação de um caderno temático no NotebookLM, foram reunidas fontes abertas e institucionais sobre Fundos de Investimento Imobiliário (FIIs). Com base nesse material, foram definidos objetivos de estudo, elaboradas perguntas estratégicas e testadas variações de prompts para organização, síntese e consolidação do conhecimento.

## Objetivo do Projeto
Construir um miniguia introdutório sobre Fundos Imobiliários, reunindo conceitos fundamentais, funcionamento, riscos, vantagens e termos essenciais, além de um conjunto de prompts reutilizáveis para apoio em revisões futuras.

## Ferramentas Utilizadas
- NotebookLM
- Inteligência Artificial Generativa
- GitHub
- Markdown

## Resultados Esperados
- Curadoria de fontes abertas e confiáveis sobre FIIs
- Resumos estruturados dos principais conceitos
- Glossário financeiro aplicado ao tema
- Conjunto de prompts reutilizáveis para estudo
- Organização do conhecimento com apoio de IA

Este repositório possui caráter educacional e demonstra a aplicação da IA como suporte ao aprendizado crítico e estruturado em finanças.
